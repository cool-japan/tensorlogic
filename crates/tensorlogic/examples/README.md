# Tensorlogic Examples

This directory contains examples demonstrating Tensorlogic usage.

## Examples

### 00_minimal_rule

A minimal example showing how to:
- Define logic expressions using `TLExpr`
- Compile rules to einsum graphs
- Run the basic planning flow

Run with:
```bash
cargo run --example 00_minimal_rule
```
